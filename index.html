Buffalo

Programming Language

Buffalo is an esoteric programming language inspired by William J. Rapaport's research into grammatically correct yet complicated linguistic constructs. Buffalo is syntactically correct, but semantically meaningless - so your colorless green ideas can sleep furiously!

Buffalo has only one primary token and was created as a way to bring semantic meaning to the grammatically correct, yet semantically meaningless sentence "Buffalo buffalo Buffalo buffalo buffalo buffalo Buffalo buffalo" concocted by William J. Rapaport.
